## List-First & Time-Tracking Rule (絶対遵守)
1. **計画の立案と登録 (List-First)**: いかなるコードの実装や作業を始める前にも、必ず「やり切るための計画」を立ててください。
2. **UIへの反映 (Put to UI/DB)**: 立てた計画は、単なるテキストのメモや回答として終わらせず、必ず Yarikiru アプリケーション（UI/UXに表示されるDB）に「プロジェクト」「中目標」「サブタスク」として登録してください。
3. **時間計測 (Time Tracking)**: タスクの作業を開始する際は、必ず MCP ツールなどを駆使してそのタスク（目標）を作業中状態にし、時間を計測・トラッキングできるようにしてください。
4. **Agentic Workflow**: 「計画登録 → 作業開始（時間計測） → サブタスク実行・完了 → 目標完了」のサイクルを遵守し、勝手に直接コードを書き始めないでください。
